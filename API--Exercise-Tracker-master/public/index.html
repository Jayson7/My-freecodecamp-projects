Example of usage

  • Use:

    "http://warm-bayou-72727.herokuapp.com/api/exercise/log?userid=E9i23rjcc"

    • To get all logs registered for this user

  • Use:

    "http://warm-bayou-72727.herokuapp.com/api/exercise/log?userid=E9i23rjcc&fromdate=1996-07-23&todate=2010-01-01"

    • To get all logs registered for this user in the range between "fromdate" to "todate"

  • Use:

    http://warm-bayou-72727.herokuapp.com/api/exercise/log?userid=E9i23rjcc&fromdate=1996-07-23&todate=2010-01-01&limit=2

    • To get all logs registered for this user in the range between "fromdate" to "todate" and "limit" the amout of logs you will recive
